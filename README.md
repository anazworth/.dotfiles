# .dotfiles

This is how I maintain a portable system for my development environment using Nix and GNU Stow.

## Instructions

1. Clone this repo into your home directory.

2. Run the `install.sh` script. (If ZSH is not preinstalled, uncomment the relevant lines in the script.)

## Vim & Neovim

* Run `:PlugInstall` when opening Vim or Neovim for the first time after running the install script.

## Ideavim

### Jetbrains plugins required

* Ideavim plugin

* Which-Key plugin (Optional, but recommended)

* EasyMotion plugin (Optional)

* AceJump plugin (Optional, but required for EasyMotion)

## ZSH

* The 'powerlevel10k' setup will run on it's own after starting a new zsh shell.
